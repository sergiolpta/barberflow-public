# BarberFlow

Plataforma de gestão para barbearias com foco em agendamento, operação e automação.

## Visão Geral
BarberFlow é uma aplicação web que centraliza o gerenciamento de barbearias, permitindo
controle de profissionais, serviços, agenda e integrações automatizadas.

## Principais Funcionalidades
- Gestão de profissionais
- Catálogo de serviços e pacotes
- Agendamento inteligente
- API REST para integrações
- Autenticação e controle de acesso
- Estrutura multi-barbearia (multi-tenant)

## Arquitetura
- Backend: Node.js + Express
- Banco de dados: Supabase (PostgreSQL)
- Infraestrutura: Docker + Traefik
- Automação: n8n (opcional)
